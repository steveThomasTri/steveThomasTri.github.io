Part 9: Finally...Time for Some More Predictions

Now let's evaluate the accuracy when we run some data through different models.

My testing data consists of 307 people from the 2019 NFL Draft. Again, I need to split the data into ten groups and put those through the different models. Some of the positions on players were changed to put them into the ten groups. For example, EDGE runners were lumped into the Defensive Lineman, because they are located near the defensive line. After splitting up the testing data, here are the counts for each group

Observe the All Test Cases folder, this is where we split the 2019TestingData.csv file into ten sets of testing data. Each of them has different features based on the different models. In the jupyter notebook, for each group, we put the models through to make sure that we get the accuracy from the previous section. Using the testing data from each group, we scrap the necessary data based on the model. After that, we run the testing data through the model to get the predictions for the players.
