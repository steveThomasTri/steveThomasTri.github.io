2019 NFL Draft

Part 1: Introduction

Answering questions as to why I am machine learning the NFL Draft

The one thing that I have learned throughout this process is to be ok with your findings. It was eye opening to witness how much rigor goes into solving a problem.

Steve Thomas 2019

Approximately two months ago, I was and still am fascinated about the world of machine learning. I had looked through tutorials about using the MNIST data set to predict numbers, classes of flowers, whether a picture was a muffin or a puppy. Going through these tutorials I had no problem in trying to set up a neural network or a decision tree or a logistic regression. However, the concept that was not clear to me was if you have set up a model with a certain accuracy, how do you then improve on that accuracy?

Then I went into exploring machine learning packages from python that you can use. Many packages are out there like sci-kit learn, tensorflow, keras, xgboost, etc. I knew that I wanted to do a classification problem, basically answering a Yes or No question, but which one would I want to do? What problem was most interesting to me?

A little background experience of the python technologies that I have used. I used BeautifulSoup, selenium/splinter for automation, flask for backend development. For data visualization, jupyter notebook, pandas, matplotlib. For machine learning, keras, sci-kit learn, tensorflow. It was time to put these to use and see if I can find a problem and attempt to solve. In our class, there was a lot of football talk about the Cleveland Browns this year and how they could be a playoff team. So I was thinking, how team select players and what is the process that prospects go through. There are "analysts" and "experts" that predict where would players go, when would they go off the board in a draft, what round are they picked. Although these questions are great for an advanced model, I just want to predict whether a player would get drafted. And so, the data science problem was stated.

Based on an NFL draft prospects height, weight, and the results that they posted in the NFL combine, will this person be drafted?

As you read through this article, pay special attention to green boxes, because over the course of two months, the main question changes as I learn new methods and ways to represent the data. Now that I have the question at hand, how would we retrieve the data?
